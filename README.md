Gypsy Resto Café — Concept Website

Overview

This project is a conceptual website design created to demonstrate a modern, minimal digital menu and brand presence for a café or restaurant. The website focuses on clarity, visual flow, and mobile-friendly interaction.

This project is intended for demonstration and portfolio purposes only.

Purpose:
Showcase a clean café website layout
Demonstrate a digital menu experience
Present responsive navigation and modern UI elements
Serve as a sample project for similar hospitality businesses

Features:
Responsive navigation with mobile menu
Hero section with image marquee
Structured digital menu layout
Clean typography and spacing
About and location section
Footer with operating hours and contact details
Technologies Used
HTML5
CSS3
JavaScript (Vanilla)
Google Fonts (Plus Jakarta Sans)
